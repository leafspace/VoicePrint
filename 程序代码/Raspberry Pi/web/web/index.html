Secure Function Lock

Secure Function Lock lets you restrict Public access to certain machine functions. This feature also prevents users from changing the Default settings of the machine by limiting access to the Menu settings. Access to restricted operations can be enabled by creating a restricted user/group. Restricted users/groups must enter a password to use the machine.
